# Project Vocabulary

Use this file to define canonical product and codebase terms for AI agents.

## Terms

| Term | Meaning | Avoid |
| --- | --- | --- |
| Content | The text a user submits for analysis (copy, post, page, doc). | input, document, article (when meaning the submission) |
| Analysis | One full SEO evaluation run over a piece of content. | scan, check (as a noun) |
| Finding | A single SEO issue with its explanation and suggested fix. | result, error, warning |
| SEO score | An optional overall rating derived from findings. Secondary to findings. | grade, rank |
| Target keyword | The keyword/topic the content should rank for; user-given or inferred. | focus word, key phrase |
| Meta description | The page's `<meta name="description">` summary used by search engines. | meta tag (too broad) |
| On-page SEO | Optimization of the content itself (keywords, structure, meta), the tool's scope. | technical SEO (out of scope) |
| Readability | How easy the content is to read; one SEO dimension among several. | grade level |
